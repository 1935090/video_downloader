FEATURES
Mobile App Developers

Feature list from app developers perspectives

Facebook video url extraction We are using basic techniques to extract video URLs from the fetched post; and use them to download the corresponding videos.

State management with Redux We are using redux to manage states and perform actions across the application.

Offline storage We are using redux-persist with AsyncStorage to save offline data

AdMob We also use AdMob in various places of the app, developers only need to include pre-coded ad components in screens they want to show the ads.

Redux-Saga for Data Api Handling We use redux-saga to handle all data request asynchronous processing. You will find various examples of how redux-saga is used to handle different logics in request processing.

Custom Icon React Native Vector Icons & icomoon are used to create custom icons for use in many places in the application.

Well Structured Project Project structure follows the structure initiated by Ignite - a React native CLI

Navigation Drawer / Stack navigation / Tabs navigation are used in the application

Well designed UIs for both OS/Android The application contains many beautiful UI components/styles for both OS and Android.

App Users

Feature list from app users perspectives

Download facebook videos Users can download facebook videos via their post URLs in HD (High Definitions) or SD (Standard Definitions).

Save videos Users can save videos inside the application or save them to Camera Roll.

Share post URLs Users can share post URLs to other applications or to users' friends.

Remove video Users can remove downloaded videos
